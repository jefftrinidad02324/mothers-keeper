Mother's Keeper: The Resilience of Apol

How to upload to GitHub Pages:
1. Extract this ZIP file.
2. Upload the extracted contents to your GitHub repository.
3. Make sure the files stay in this structure:
   index.html
   README.txt
   assets/cover.png
   videos/apol.mp4
   videos/mother.mp4
   videos/sister.mp4
   videos/coworker.mp4
4. In GitHub, click Commit changes.
5. Go to Settings > Pages.
6. Source: Deploy from a branch.
7. Branch: main. Folder: /(root).
8. Save and wait for the website link.

Note: No style.css is needed because the CSS is already inside index.html.
